An online shop API, built using Node, Express and MongoDB

Schemes: http, https
Version: 1.0.0

Users

Manipulate users

Users

GET /users

Returns all users from the API

200 OK

Response when there were no errors

500 Internal Server Error

Response when something went wrong on the API side

Users

POST /users

Creates a new user and return it to the client

201 Created

Response when there were no errors

400 Bad Request

Response when user already exists in the database

500 Internal Server Error

Response when something went wrong on the API side

Users

PUT /users

Updates the user identified by the provided authorization token and return it to the client

Authorization: object
in header

The authorization token

201 Created

Response when there were no errors

401 Unauthorized

Response when no authorization token was provided

404 Not Found

Response when user doesn't exist in the database

500 Internal Server Error

Response when something went wrong on the API side

Users

Sessions

Manipulate sessions

Sessions

POST /sessions

Creates a session for the user with the given credentials and returns it to the user

201 Created

Response when there were no errors

404 Not Found

Response when the user with the specified credentials doesn't exist in the database

500 Internal Server Error

Response when something went wrong on the API side

Sessions

DELETE /sessions

Removes the session specified in the authorization header

Authorization: object
in header

The authorization token

200 OK

Response when there were no errors

401 Unauthorized

Response when no authorization token was provided

500 Internal Server Error

Response when something went wrong on the API side

Sessions

GET /sessions/{provider}/oauth

provider: object
in path

Uses the specified OAuth provider

code: object
in query

The request code provided by the provider

200 OK

Response when there were no errors

400 Bad Request

Response when no request code was provided or the request code was invalid/expired

500 Internal Server Error

Response when no authorization token was provided
